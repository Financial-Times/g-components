# Navigation header component

`<NavigationHeader />` can be used to add extra navigation to a page. Includes a dropdown for small devices

Props:

- `title` (string - optional): navigation title to be displayed to the left of the links
- `links` (array): a list of links for navigation each with a `text` and `url` attribute
- `currentPage` (string - optional): if this matches the url of an item in the links array, this will
  be highlighted in the navigation
- `useDropdown` (boolean - optional): an override to enable/disable the dropdown. If this prop is undefined, the
  dropdown is automatically displayed at the `default` breakpoint
- `openLinksInNewTab` (boolean - optional): adds `target="_blank"` to all links so they open in another tab
